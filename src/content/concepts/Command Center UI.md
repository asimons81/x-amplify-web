# Command Center UI Vision

The Second Brain needs to evolve from a simple file viewer into a **Content Command Center**.

## **Visual Direction**
- **Cyber-Tech Aesthetic:** Dark background (zinc-950), high-contrast borders (zinc-800).
- **Glassmorphism:** Use heavy backdrop-blur (20px+) for all control panels.
- **Accents:** Neon blue (#3b82f6) for "AI Activity" and Amber for "Action Required".
- **Typography:** Inter for body, and a mono font (JetBrains Mono) for data/stats.

## **Core Modules**
1. **Trend Pulse:** Real-time stream of ideas from Trend Scout.
2. **The Forge:** Remotion render queue status + preview thumbnails.
3. **Ghostwriter:** One-click approval for drafted social posts.
4. **Exit Tracker:** Financial milestones vs ISU income replacement.

## **Interactions**
- Subtle hover glows.
- Micro-animations for data updates.
- Spring-based entrance for all modal windows.
